# English Grammar Hub

*English Grammar Hub* is an educational website designed to help students, beginners, and English learners understand grammar in a simple and easy way. The site provides clear explanations, examples, and practice exercises for various grammar topics.



# Features

  **Parts of Speech**  
  Learn about Nouns, Pronouns, Verbs, Adjectives, Adverbs, Prepositions, Conjunctions, and Interjections.

  **Tenses**  
  Understand all English tenses with examples and usage.

  **Articles**  
  Explanation of "a," "an," and "the" with examples.

  **User-Friendly Design**  
  Simple and modern interface suitable for all learners.



## Pages

- `index.html` – Home page  
- `pos.html` – Parts of Speech  
- `tens.html` – Tenses  
- `artical.html` – Articles  
- `activ.html` – Active & Passive Voice  
- `login.html` – (Optional login page for future features)
